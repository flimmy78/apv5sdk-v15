#!/usr/bin/haserl Firewall
Firewall include NAT and URL filtering. The implement of NAT have three ways: Static Nat, Dynamic Nat and port multiplexing OverLoad. Port multiplexer (Port address Translation, PAT) is to change the source port and port conversion when the packet is going out, device is configuring such NAT, the default interface for eth0. URL filtering is adding the URL filtering rules to device to achieve the filtering.
