#!/usr/bin/haserl 3G Setting
The 3G page allows a user to configure 3G usb interface of the AP device. When users insert 3G card to 3G usb interface of the device, the device can link in 3G network by dial-up in 3G network environment. Here is mainly PPP dial-up configuration parameters. PPP Connect Mode: Divided into automatic connection, on-demand connections and manually connect Dial-up number: The default is * 99 # APN access way: not support PDP type: not support PDP access way: not support PPP Authentication: not support PDP type: not support Setting idle time £º Maximum Transmission Unit £º DNS access: include active and static, static access can be prepared with the main DNS server
