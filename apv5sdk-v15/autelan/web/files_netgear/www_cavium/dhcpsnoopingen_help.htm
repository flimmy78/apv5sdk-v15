#!/usr/bin/haserl DHCP Snooping
DHCP-Snooping allows a physical port is set to trust or no-trust port. Trust ports can receive and transmit the normal dhcp offer message,but the no-trust ports will receive the dhcp offer packet discarded. In this way, ensuring that dhcp client from the legitimate dhcp server to obtain a IP address. option82 is one of dhcp relay agent information option. when dhcp client sent the request message to the dhcp server, if the need to go through dhcp relay, dhcp relay will add the option82 string to the message of the request. With option82,dhcp server can record the information of dhcp relay and dhcp client, and implement the limit of dhcp distribution and the Accounting function.
