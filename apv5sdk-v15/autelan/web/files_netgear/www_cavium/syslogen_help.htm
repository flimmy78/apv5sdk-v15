#!/usr/bin/haserl Syslog
The Syslog page allows a user to get the syslog information from WTP. Click the button will update the information shown on the page. Click the button, and then the user can save the syslog on local disk. Enable Sys Log Server: Enable or disable a syslog function. If the user checks the checkbox, then the WTP can send syslog information to Syslog server. Server IP: Input the syslog server IP address. Server port: Input the port number used by the syslog server. After complete all configuration, click < Apply> button to validate modification. If click the button, and set the upload server IP and time interval, then click the button, the syslog info will be uploaded to server at regular time.
