#!/usr/bin/haserl DHCP Server
establish a DHCP server, to distribute IP addresses to the wireless terminal who accessed 1.Enable DHCP server state: whether to activate the DHCP server 2.DHCP lease: IP address assigned to the effective time. Access to users over the lease after the previous IP address will be invalid, need to re-apply to the IP address of AP. 3.Subnet Mask: DHCP server IP address to the requesting users to specify the subnet mask. 4.Gateway: DHCP server IP address to the requesting user-specified gateway. 5.Ip address range: that is, address pool, DHCP assigned IP address of the non-existence can only be in this range
