#!/usr/bin/haserl Back Setup
The Back Setup page allows a user to save current configuration on WTP, or upload a configuration file to WTP. It can also revert to factory default setting. Save a copy of current setting: save current configuration. Click the button, then click and save the configuration as a XML file. Restore saved setting from files: Select a configure file on the disk, and then click button. Revert to factory default settings: Click the button, and then default configuration on WTP will be recovered.
