#!/usr/bin/haserl Diagnostics
The Diagnostics page allows a user use some common tools to check whether the network runs well. Ping: Input an IP address, and then click the button. After that, the page will return the result of ping several seconds later. Nslookup: Input a domain name, and then click the button. After that, the page will return the result of nslookup several seconds later. Traceroute: Input a domain name or IP address, and then click the button. After that, the page will return the result of traceroute several seconds later.
