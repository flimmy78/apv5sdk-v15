#!/usr/bin/haserl DNS Server
set DNS server's address . you can choose "Get DNS from DHCP" or "Use These DNS" to set IP address 1. Get DNS from DHCP: get DNS server address by DHCP; in this pattern , you should make sure that DHCP server can offer DNS server address. 2. Use These DNS: DNS server 's address set by yourself. you could set "first DNS" and "backup DNS" to address. all this value should be input in dotted-decimal format, which like xxx.xxx.xxx.xxx
